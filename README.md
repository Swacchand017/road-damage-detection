# Road Damage Detection System

##  About the Project

This project is a simple and practical **Road Damage Detection System**.
It uses a deep learning model to identify road issues like **cracks and potholes** from images and videos.

I built this project using **Python, TensorFlow, and OpenCV**, and added a **GUI interface** so it’s easy to use without coding knowledge.

---

##  What this project does

* Detects road damage from video files
* Uses a trained deep learning model
* Provides a simple graphical interface
* Works on normal CPU (no GPU required)

---

##  Technologies Used

* Python
* TensorFlow / Keras
* OpenCV
* NumPy, Pandas
* Matplotlib, Seaborn
* Tkinter (for GUI)

---

##  How to Run

1. Clone this repository:

```
git clone https://github.com/Swacchand017/road-damage-detection.git
cd road-damage-detection
```

2. Install required libraries:

```
pip install tensorflow==2.10.0 keras==2.10.0 numpy==1.23.5
pip install opencv-python pandas matplotlib seaborn scikit-learn
```

3. Run the project:

```
python Main.py
```

---

##  How to Use

* Click on **Select Video File**
* Choose a road video
* Click **Start**
* The system will detect damages in the video
